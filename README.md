# WebRTC Audio & Video Streaming App

A Streamlit application that uses WebRTC to stream audio and video with low latency.

## Installation

Install dependencies using Poetry:

```bash
poetry install
```

## Running the App

Run the app on port 3000:

```bash
poetry run streamlit run app.py --server.port 3000
```

## Features

- Real-time audio and video streaming using WebRTC
- Low latency with synchronized audio and video
- Responsive UI with async processing
